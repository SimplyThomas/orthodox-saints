# Relationship Network — Changelog

A running changelog of the saint relationship-network project.

**Cadence:** at the end of **every 100 completed saint profiles** (each time the work crosses
an OS-#### hundred boundary — OS-0100, OS-0200, OS-0300, …), a changelog entry is generated
here **before moving on to the next batch**.

- Entries are concise summaries (what changed, templates promoted, groups created, corrections,
  validation status). Deeper analysis lives in the milestone reports (`docs/relationship-milestone-NN.md`).
- File naming: `NNN-os-XXXX-YYYY.md`, zero-padded and ordered (e.g. `001-os-0000-0100.md`).
